# Welcome to LocalSearch 🔍

LocalSearch is a **high-performance, private search engine** for your local files. Everything runs locally — your data never leaves your machine.

## Features
- ⚡ Instant full-text search with fuzzy matching
- 📁 Recursive directory indexing
- 🎯 File type filters and sorting
- ✏️ In-browser file editing
- 📤 Drag-and-drop file uploads
- 🗂️ Duplicate detection & cleanup
- 🔓 Open files in native applications

## Getting Started
1. Place your files in the `notes/` directory (or configure a custom path in Settings)
2. Run `npm run dev` to start the server
3. Open `http://localhost:3000` and start searching!

## Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Focus search bar |
| `Escape` | Close preview |
